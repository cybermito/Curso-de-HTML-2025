# Curso de HTML actualizado a 2025

En este curso se verá todos los fundamentos de HTML y como aplicarlo. Cada carpeta contiene los ejemplos de cada clase y también toda la teoría resumida.

## Contenido del curso

I. Fundamentos de HTML

---

    1. Maquetación web con HTML5 y accesible.
    2. Cómo funcionala web desde URL hasta página renderizada.
    3. Configuración de VSCode y estructura HTML5 básica.
    4. Etiquetado de texto y jerarquía en HTML.
    5. Insertar imágenes en HTML con optimización y accesibilidad.

II. Listas y Enlaces

---

    1. Listas ordenadas, desordenadas y enlaces en HTML
    2. Etiquetas semánticas en HTML5

III. Formularios y Tablas

---

    1. Formularios en HTML5 con form, label, input y validación.
    2. Tipos de inputs para formularios avanzados.
    3. Checkbox y select en formularios HTML.
    4. Validaciones de formularios en HTML.
    5. Estructura de tablas HTML con thead, tbody y tfoot.

IV. Contenido Multimedia

---

    1. Imágenes responsivas en HTML5 con la etiqueta picture y srcset para mejorar el rendimiento.
    2. Integración de vídeo y audio en HTML5 con controles y compatibilidad.

V. Accesibilidad Web y SEO.

---

    1. Atributos ARIA para accesibilidad web con lectores de pantalla.
    2. Estructura Semántica y meta tags para mejorar posicionamiento web.

VI. Proyecto Final

---

    1. Creación de un portafolio profesional con HTML semántico y SEO

## Requisitos

- Conocimientos básicos de informática
- Un editor de texto (como Visual Studio Code, Sublime Text, etc.)
- Un navegador web (como Google Chrome, Mozilla Firefox, etc.)
- Ganas de aprender y practicar
